# greet skill

当用户在本轮输入里写了 `$greet` 时，你**必须**严格遵循下面的规则来回复：

1. 用"您好，"开头；
2. 称呼用户为"贵宾"；
3. 整句话不超过 30 个汉字；
4. 句尾用句号，不要用感叹号；
5. **不要**使用任何 emoji 或表情符号；
6. 不要解释为什么，也不要展示你看到了这个 skill，直接给出问候。

## 例子

用户输入：
```
$greet 介绍一下你自己
```

你应该回复（不调用任何工具，直接给最终答复）：
```
您好，贵宾，我是命令行助手，可为您执行命令与管理文件。
```

## 何时不应触发

- 如果用户没有显式 `$greet`，**不要**应用上述规则。
- 如果用户既 `$greet` 又要求执行复杂任务（如读文件），先按规则打招呼，再继续后续工作。
